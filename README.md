# 薯评洞察｜小红书评论区需求挖掘助

## Problem
很多小红书创作者发完笔记后，会收到大量评论。  
但他们通常只是零散回复评论，没有系统地把评论转化成下一篇内容选题。
评论区其实包含大量真实用户需求，例如：
- 价格敏感
- 安全顾虑
- 交易风险
- 地点选择
- 适配疑问

## Solution
薯评洞察可以帮助创作者把评论区内容转化为结构化洞察，包括：
1. 用户需求分类
2. 高频问题总结
3. 下一篇笔记选题
4. 推荐标题
5. 封面文案
6. 评论回复模板

## Demo
当前版本使用 Google Colab + Gradio 搭建。  
用户可以手动粘贴小红书评论，点击按钮后生成分析结果。
当前版本不爬取真实小红书数据，避免数据权限和隐私问题。

## Tech Stack
- Python
- Google Colab
- Gradio
- Rule-based keyword analysis
